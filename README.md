# 📖 Overview

**SmartSentry** is a Machine Learning-based Cyber Threat Detection System designed for securing **Industrial Internet of Things (IIoT)** environments.

The application analyzes network traffic using multiple Machine Learning and Deep Learning algorithms to identify malicious activities and classify cyber threats with high accuracy.

The project also provides a **Flask-based web application** that allows users to upload network data and receive real-time threat predictions.

# 📚 Research Motivation

Industrial IoT systems are increasingly becoming targets for cyber attacks due to their widespread adoption in manufacturing and critical infrastructure. Traditional security mechanisms often fail to detect sophisticated attacks in real time.

This project applies Machine Learning techniques to analyze network traffic patterns and identify malicious behavior, enabling faster and more accurate cyber threat detection while improving the overall security of IIoT environments.
